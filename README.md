[Memory](http://gfcarvalho.github.io/memory-game)
===================

By Gustavo Carvalho

An old fashion HTML5 memory game.

##Features
- Only easy mode with 4x4 grid available for now.

##Future
- More game modes
- Store best scores
- Challenge mode (increasing difficult after each game)
- ... and much more
